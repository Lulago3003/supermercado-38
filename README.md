# Supermercado 38

Sitio web comercial de varias páginas para Supermercado 38, Arraiján, Panamá Oeste.

## Páginas

- Inicio
- Ofertas de la semana
- Departamentos
- Panadería
- Contacto y ubicación

Es un sitio estático: puede visualizarse abriendo `index.html` en cualquier navegador o publicarse directamente desde GitHub Pages.
